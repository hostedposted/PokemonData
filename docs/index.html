API Quickstart#back to top

Get a random Pokemon!

Using the language / framework of your choice:

  • Load https://thepokeapi.com/pokemon/random
  • Parse the response body
Try that or some other stuff below:

Confused? You can try pokemon/pikachu, pokemon/300, type/1, type/fairy, pokemon/random?min=100&max=500, etc.

GET Methods#back to top

For all of these {endpoint} is the api endpoint, {dexId} is the id of the item, {name} is the items name.

/{endpoint}/all

This will return all info under given endpoint.

/{endpoint}/random

This will randomly return a json for the given endpoint. You also can change the min id and the max id by using /{endpoint}/random?min=10&max=20 changing min to the minimum and max to the maximum. Or just /{endpoint}/random?max=20 not giving the min parameter this will also work vice-versa.

/{endpoint}/{name}

This will return the json of the given name from the given endpoint.

/{endpoint}/{dexId}

This will return the json of the given dexId from the given endpoint.

Usage Examples#back to top

Node.js (JavaScript)

Using npm fetch

  • Install fetch using npm (or yarn)
    • npm i fetch
    or 
    yarn add node-fetch
  • Create a file ThePokeApi.js
    • 
const fetchUrl = require("fetch").fetchUrl;

const url = 'https://thepokeapi.com/pokemon/random';

fetchUrl(url, function(error, meta, body) {
    if (!error) {
        var data = new String();
        data = JSON.parse(body)
        // Do something with data
        console.log(data.name)
    }
});
  • Run the file using the following code (in the terminal)
    • node ThePokeApi.js
  • You will see the following output in the terminal in this case
    • froakie

Python

  • Install requests using pip or any other package manager
    • pip install requests
  • Create a file ThePokeApi.py
    • 
import requests

response = requests.get("https://thepokeapi.com/pokemon/random").json()

print(response['name'])
  • Run the file in your IDE or by running the following code
    • python -u "{pathToFile}\ThePokeApi.py"
  • You will see the following output in this case
    • cufant

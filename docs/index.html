Usage Examples#back to top

Node.js (JavaScript)

Using npm fetch

  • Install fetch using npm (or yarn)
    • npm i fetch
    or 
    yarn add node-fetch
  • Create a file ThePokeApi.js
    • 
const fetchUrl = require("fetch ").fetchUrl;

const url = 'https://thepokeapi.com/pokemon/random';

fetchUrl(url, function(error, meta, body) {
    if (!error) {
        var data = new String();
        data = JSON.parse(body)
        // Do something with data
        console.log(data.name)
    }
});
  • Run the file using the following code (in the terminal)
    • node ThePokeApi.js
  • You will see the following output in the terminal in this case
    • froakie

Python

  • Install requests using pip or any other package manager
    • pip install requests
  • Create a file ThePokeApi.py
    • 
import requests

response = requests.get("https://thepokeapi.com/pokemon/random").json()

print(response['name'])
  • Run the file in your IDE or by running the following code
    • python -u "{pathToFile}\ThePokeApi.py"
  • You will see the following output in this case
    • cufant
